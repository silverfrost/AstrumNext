0.80.2 2551010.  Search and replace is much more robust. In addition the zero replacement   size warning has been removed.  Cut and delete operations would result in the cursor moving to the end   of the document. It no longer does this.  Join now moves the cursor to the first line of the joined text. In additional   it handles the case where the text in the joined file uses a different EOL   marker.  Save As now functions correctly when the new file name matches an existing file.  Astrum respects the files directory rather than always defaultin   to its own directory.  Three source files added. These are included in the Zeus for Next release and   have been kindly donated by Simon N Goodwin (https://simon.mooli.org.uk/nextech/z80n/index.html)      fpcodes.asm  Floating point operator codes      nextreg.asm  Next register names      sysvars.asm  Next system variables.  Starting from a fresh install pressing ENTER P UP UP produces 'a mess'. This   has been tidied.  Debugger has new commands:     e xxxx nn  sets the byte at xxxx to nn     ew xxxx nnnn  sets the word at xxxx (and xxxx+1) to nnnn     er rr nnnn  sets the register pair rr to nnnn     i nn  display a read from the Next register nn.  Debugger will now show alternate registers with the r command (and they can be use with er)0.80.1 2550607.  ChangeHistory added.  Some delete block bugs fixed in the editor. These are when the marked   block includes the last line of the text file..  .z80 and .pas file extensions added to list of those editable..  Some code removed at the suggestion of the assembler directive   LIST UNUSED.0.8 2506011630. Some issue with the debugger have been resolved,   in some situations it would not run at all.0.8 2505301827.  Initial release onto GitHub.
